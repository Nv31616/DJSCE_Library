# DJSCE_Library
A basic website built using Django &amp; Tailwind CSS to search for books in DJSCE Library and the get its location on a map visual.

## How to visit
It is deployed on vercel.
Use this link [https://djsce-library-mq8y.vercel.app/]

## Tech Stack
1. **Frontend**: HTML | Tailwind CSS | JavaScript
2. **Backend**: Django
3. **Database**: Neon(PostgreSQL) (Vercel default)

More improvements are on the way for the website!
